## 5. Diretrizes de Uso

### 5.1 Logotipo

- O logotipo principal (`logo-divino-coracao.png`) deve ser usado preferencialmente sobre fundo claro.
- A versão em preto e branco (`logo-divino-pb.png`) é para materiais monocromáticos.
- Não recolocar, distorcer ou aplicar efeitos sobre o logotipo.
- Manter área de respiro mínima de 10% da altura do logotipo em cada lado.

### 5.2 Cores

- A cor primária `#81211F` (bordô) é a cor de assinatura institucional — deve predominar.
- O amarelo `#FFCC2A` é a cor de realce — usar com moderação para chamar atenção.
- Texto corrido deve usar `#333` sobre fundo claro para garantir legibilidade.
- Títulos devem usar `#1a1a1a`.
- Fundo escuro (`#222` ou `#1a1a1a`) deve ter texto claro (`#ccc` ou `#fff`).

### 5.3 Tipografia

- A fonte Ubuntu é a fonte corporativa para corpo de texto e botões.
- Para materiais que não puderem usar Ubuntu (ex.: Google Docs), usar como fallback a fonte **sans-serif** padrão do sistema (Arial, Helvetica).
- Headings podem usar Ubuntu ou Roboto, ambos em peso 400 (Regular).
- Não usar itálico em blocos longos de texto.

### 5.4 Botões

| Propriedade | Valor |
|------------|-------|
| Fonte | Ubuntu 700 (Bold) |
| Tamanho | 16px |
| Padding | 1.0em 2.0em |
| Borda | 0 |
| Border-radius | 0.3em |
| Fundo padrão | `#81211F` |
| Texto | `#ffffff` |
| Fundo hover | `#81211F` (mantido, sem variação) |

### 5.5 Ícones

- Usar a biblioteca **Font Awesome 5** para consistência.
- Ícones sociais devem usar a família `fab` (Brands).
- Ícones funcionais do site usam `fas` (Solid).
- Tamanho padrão de ícones inline: 1em.
